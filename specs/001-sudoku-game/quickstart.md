# Quickstart: Modern Sudoku Game Website

## Prerequisites

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Run in Development

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Test

```bash
npm run test
```

## Notes

- This is a static site intended for GitHub Pages deployment.
- No database or server-side components are required.
